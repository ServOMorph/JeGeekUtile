# Geekos - Monnaie Virtuelle Je Geek Utile

## Presentation

Les **Geekos** sont la monnaie virtuelle de l'association Je Geek Utile. Cette monnaie recompense l'engagement des benevoles et peut etre echangee contre des avantages au sein de l'association.

## Fonctionnement

### Obtention de Geekos

Les benevoles obtiennent des Geekos en :

- **Realisant des missions** : chaque mission accomplie rapporte des Geekos selon sa difficulte et sa duree
- **Participant aux evenements** : presence aux reunions, formations, ateliers
- **Contribuant au developpement** : propositions d'amelioration, creation de contenu
- **Parrainant de nouveaux benevoles** : bonus pour chaque nouveau benevole actif parraine

### Bareme indicatif

| Action | Geekos |
|--------|--------|
| Mission courte (< 2h) | 10-30 |
| Mission moyenne (2-5h) | 30-80 |
| Mission longue (> 5h) | 80-200 |
| Participation evenement | 20 |
| Formation suivie | 50 |
| Formation animee | 100 |
| Parrainage actif | 50 |

### Attribution

Les Geekos sont attribues par les administrateurs via l'interface d'administration du site. L'administrateur peut :

- Consulter le solde de chaque benevole
- Ajouter des Geekos apres validation d'une mission
- Retirer des Geekos en cas d'echange

## Utilisation

### Avantages disponibles

Les Geekos peuvent etre echanges contre :

- **Formations premium** : acces a des formations avancees
- **Materiel** : pret de materiel informatique
- **Goodies** : produits derives de l'association
- **Priorite** : acces prioritaire aux evenements a places limitees
- **Reconnaissance** : badges et distinctions sur le profil

### Procedure d'echange

1. Le benevole consulte son solde de Geekos dans son profil
2. Il contacte l'administration pour demander un echange
3. L'administrateur valide et deduit les Geekos du solde

## Regles

- Les Geekos ne sont pas convertibles en argent reel
- Les Geekos sont personnels et non transferables entre benevoles
- Le solde de Geekos n'expire pas
- En cas de depart de l'association, le solde est remis a zero
- L'administration se reserve le droit de modifier le bareme

## Affichage

Le solde de Geekos est visible :

- Dans la barre de navigation (pour les utilisateurs connectes)
- Dans le profil du benevole
- Dans l'interface d'administration (pour les admins)

Le logo Geekos accompagne toujours l'affichage du solde pour une identification rapide.

## Contact

Pour toute question concernant les Geekos, contacter l'administration via le formulaire de contact du site.
