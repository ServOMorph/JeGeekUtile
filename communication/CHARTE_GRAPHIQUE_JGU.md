# 2. LOGO

## 2.1 Logo principal

Le logo combine trois elements :

1. **Symbole** : Silhouette humaine stylisee entouree d'un halo lumineux (cercle ouvert)
2. **Typographie** : "Je Geek Utile" en Poppins Semi-Bold
3. **Accent** : Point lumineux (couleur or) sur le "i" de Utile

**Construction**
```
    .-~~~-.
   /  o    \      <- Halo technologique (cercle ouvert)
  |   /\    |     <- Silhouette humaine simplifiee
  |  /  \   |
   \______/

   Je Geek Utile
         ^
         |
      Point or
```

## 2.2 Versions du logo

| Version | Usage |
|---------|-------|
| Logo complet (symbole + texte) | Communications officielles, en-tetes |
| Logo horizontal | Banniere web, signatures email |
| Symbole seul | Favicon, icone app, petits espaces |
| Logo monochrome blanc | Fonds sombres |
| Logo monochrome noir | Impressions N&B |

## 2.3 Zone de protection

Espace minimum autour du logo : hauteur du symbole divisee par 4 sur tous les cotes.

## 2.4 Taille minimale

- Logo complet : 120px de large (digital) / 30mm (print)
- Symbole seul : 24px (digital) / 8mm (print)

## 2.5 Interdictions

- Ne pas etirer ou deformer le logo
- Ne pas changer les couleurs hors versions prevues
- Ne pas ajouter d'effets (ombre portee, biseaux)
- Ne pas placer sur fond trop charge
- Ne pas entourer d'un cadre
- Ne pas modifier la typographie

# 4. TYPOGRAPHIE

## 4.1 Polices officielles

### Police principale : Poppins

**Famille** : Sans-serif geometrique arrondie
**Source** : Google Fonts (gratuite)
**Caracteristiques** : Moderne, accessible, chaleureuse

| Style | Poids | Usage |
|-------|-------|-------|
| Regular | 400 | Corps de texte |
| Medium | 500 | Sous-titres, emphase |
| Semi-Bold | 600 | Titres, navigation |
| Bold | 700 | Titres principaux, CTA |

### Police secondaire : Nunito

**Usage** : Textes longs, articles, contenus pedagogiques
**Caracteristiques** : Tres lisible, arrondie, amicale

### Police code : JetBrains Mono

**Usage** : Blocs de code, donnees techniques
**Caracteristiques** : Monospace, ligatures, lisible

## 4.2 Hierarchie typographique

### Web / Digital

| Element | Police | Taille | Poids | Interligne |
|---------|--------|--------|-------|------------|
| H1 | Poppins | 48px | 700 | 1.2 |
| H2 | Poppins | 36px | 600 | 1.25 |
| H3 | Poppins | 28px | 600 | 1.3 |
| H4 | Poppins | 22px | 500 | 1.35 |
| Paragraphe | Nunito | 16px | 400 | 1.6 |
| Small | Nunito | 14px | 400 | 1.5 |
| Caption | Poppins | 12px | 400 | 1.4 |

### Print

| Element | Police | Taille | Poids |
|---------|--------|--------|-------|
| Titre principal | Poppins | 42pt | 700 |
| Titre section | Poppins | 24pt | 600 |
| Sous-titre | Poppins | 18pt | 500 |
| Corps | Nunito | 11pt | 400 |
| Notes | Nunito | 9pt | 400 |

## 4.3 Regles typographiques

- Pas de texte en majuscules pour les longs contenus
- Majuscules autorisees pour : boutons, labels courts, acronymes
- Italique reserve aux citations et termes etrangers
- Soulignement reserve aux liens hypertexte
- Espacement des lettres : 0 pour corps, +2% pour titres

# 6. IMAGERIE

## 6.1 Direction photographique

**Sujets privilegies** :
- Personnes de tous ages utilisant la technologie
- Mains sur clavier/souris (proximite, action)
- Visages souriants devant ecrans
- Groupes en formation/atelier
- Details humains (mains, expressions)

**A eviter** :
- Photos de banques d'images generiques
- Technologie sans humain
- Images trop froides ou corporates
- Ecrans vides ou factices
- Poses artificiel les

## 6.2 Traitement des images

**Ton** : Chaud, leger virage vers les tons or/orange
**Contraste** : Modere, pas d'aplats de noir
**Overlay autorise** : Degrade Aurore a 60% opacity maximum

## 6.3 Illustrations

**Style** : Flat design humanise avec touches organiques
**Personnages** : Proportions realistes, diversite representee
**Couleurs** : Palette Aurore uniquement
**Details** : Coins arrondis, formes douces

- Zone de texte : 6 colonnes centrees

## 9.2 Digital

### Site web

- Header : Logo gauche, navigation droite, fond Noir Chaud
- Hero : Grande typo, CTA degrade, image avec humain
- Sections : Alternance fonds Noir Chaud / Brun Nuit
- Footer : Coordonnees, reseaux, baseline

### Reseaux sociaux

**Photo de profil** : Symbole seul sur fond Noir Chaud

- Fond : Noir Chaud
- Titres : Poppins Bold, Beige Clair
- Accent : Degrade Aurore pour elements importants

### Affiche A3 / A2

**Structure** :
1. Accroche (Poppins Bold 60pt+)
2. Visuel humain (photo ou illustration)
3. Message cle (3 lignes max)
4. Logo + coordonnees
5. QR code (optionnel)

**Fond** : Noir Chaud ou Degrade Aurore
**Texte** : Beige Clair ou Noir Chaud (si fond clair)

- Recto : Accroche + visuel + CTA
- Verso : Details + contact + QR code

### Roll-up (85 x 200 cm)

- Zone haute : Logo grand format
- Zone centrale : Message cle + illustration
- Zone basse : Coordonnees + baseline
- Fond : Degrade vertical (Noir Chaud vers Brun Nuit)

# 10. VOIX ET TON

## 10.1 Principes redactionnels

**Clarte** : Phrases courtes, mots simples, pas de jargon non explique
**Proximite** : Tutoiement ou vouvoiement selon contexte, jamais de "on"
**Positivite** : Formuler les solutions, pas les problemes
**Concret** : Exemples, chiffres, cas pratiques

## 10.2 Vocabulaire

**A privilegier** :
- Accompagner, aider, soutenir
- Apprendre, decouvrir, comprendre
- Proteger, securiser, respecter
- Simple, accessible, pour tous
- Responsable, ethique, durable

**A eviter** :
- Disrupter, revolutionner, game-changer
- Technologie de pointe, cutting-edge
- Digital native, early adopter
- Monetiser, scaler, pivoter
- Jargon tech non explique

## 10.3 Exemples de formulations

| Au lieu de... | Preferer... |
|---------------|-------------|
| "Solution innovante basee sur l'IA" | "Un outil intelligent qui vous aide a..." |
| "Optimisez votre workflow" | "Gagnez du temps au quotidien" |
| "Interface user-friendly" | "Simple a utiliser" |
| "Democratiser la tech" | "Rendre la technologie accessible a tous" |
