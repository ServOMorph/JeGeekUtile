# /halu - Détecteur d'hallucinations

## Rôle
Analyser prompt optimisé pour détecter hallucinations et incohérences

## Tâche
1. Recevoir prompt optimisé de /promptparfait
2. Vérifier cohérence logique
3. Détecter contradictions
4. Identifier extrapolations non fondées
5. Valider faisabilité technique
6. Retourner évaluation (valid/invalid + raisons)

## Critères validation
- Absence de contradictions internes
- Cohérence avec contexte projet
- Faisabilité technique vérifiable
- Pas d'extrapolations non demandées
